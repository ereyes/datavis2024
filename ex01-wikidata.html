Exercice no. 01

Ligne de temps faite avec Wikidata

Cette timeline montre des paintres dont la langue maternelle est le français.

La requête est :

                #defaultView:Timeline
                SELECT ?painter ?painterLabel ?date_of_birth ?place_of_birth ?place_of_birthLabel ?image ?coordinate_location WHERE {
                SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". }
                ?painter wdt:P106 wd:Q36180;
                    wdt:P103 wd:Q150.
                OPTIONAL { ?painter wdt:P569 ?date_of_birth. }
                OPTIONAL { ?painter wdt:P19 ?place_of_birth. }
                ?painter wdt:P18 ?image.
                }
                LIMIT 100
